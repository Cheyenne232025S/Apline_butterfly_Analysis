# Apline_butterfly_Analysis
Statistical analysis and predictive modeling of alpine butterfly populations using long-term weather and population time-series data. This project applies regression, time-series, and GLM methods to identify key weather drivers of Parnassius smintheus population dynamics and evaluate out-of-sample predictive performance.
